■ 已知量
m = 0.1 kg
q = 5.0×10⁻⁴ C
E = 2.0×10³ N/C
L = 1.0 m
g = 10 m/s²
第(1)问：求电场力和合力
①F电 = qE
②= 5.0×10⁻⁴ × 2.0×10³
③= 1.0 N
④G = mg = 0.1 × 10
⑤= 1.0 N
⑥F电与G垂直，大小相等
⑦F合 = √(F电² + G²)
⑧= √2 = 1.41 N
F电 = 1.0 N，F合 = √2 ≈ 1.41 N
第(2)问：求 θ=37° 时的速度
①h = L(1-cos37°) = 1.0×(1-0.8)
②= 0.2 m
③x = L·sin37° = 1.0×0.6 = 0.6 m
④由动能定理：
⑤W合 = F电·x − G·h
⑥= 1.0×0.6 − 1.0×0.2 = 0.4 J
⑦½mv² = 0.4 → v² = 8
⑧v = √8 = 2√2 ≈ 2.83 m/s
v = 2√2 ≈ 2.83 m/s
第(3)问：求细线最大拉力
①最低点A速度最大，拉力最大
②从释放点(θ=90°)到最低点A：
③W合 = F电·L + G·L
④= 1.0 + 1.0 = 2.0 J
⑤½mvA² = 2.0 → vA² = 40
⑥Tmax − G = mvA²/L
⑦= 0.1×40÷1.0 + 1.0
⑧Tmax = 5.0 N
Tmax = 5.0 N

✅ 最终答案汇总

(1) 电场力F电 = 1.0 N
(1) 合力F合 = √2 ≈ 1.41 N
(2) θ=37° 速度v = 2√2 ≈ 2.83 m/s
(3) 最大拉力Tmax = 5.0 N
