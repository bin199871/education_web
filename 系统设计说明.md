# 物理题目讲解动画自动生成系统 · 架构规范

## 系统概述

本系统用于将一道物理题目自动转换为HTML讲解动画。输入为题目文本，输出为可直接播放的HTML文件。

## 四层架构

### 第1层：题目分析Agent

**职责**：接收题目文本，输出结构化题目分析JSON。

**输入**：一道物理选择题（含题干和四个选项）

**输出格式**：
```json

    "key_formula": "核心公式",

      "变量名": { "name": "中文名", "unit": "单位", "property": "关键属性说明" }

第2层：幕结构匹配引擎
职责：根据题目分析结果，匹配预定义的叙事模板，输出各幕的参数配置。

幕结构模板（选择题型）：

幕次	名称	核心任务	建议时长

第三幕	总结升华	提炼核心结论，知识迁移扩展	50-60秒
第一幕固定结构：

引入（物体出场 + 概念标签）→ 10秒

场景A展示（仪器读数）→ 20秒

场景B展示（同仪器不同读数）→ 20秒

对比结论（分屏 + 核心结论弹出）→ 20秒

第二幕固定结构：

选项逐个击破 → 30秒
