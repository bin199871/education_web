按**视觉复用度**从高到低分组，同组内共享 Canvas 绘制函数和场景结构。

| 组 | 题型数 | 核心复用基础 | 优先级 |
|----|--------|-------------|--------|
| **A组** 单物体运动 | 4 | 复用电场单摆的圆弧/斜面摩擦的物块 | P0 |

## 2. A组：单物体直线/曲线运动

### A-1 竖直圆周运动（过山车/水流星）

**基于现有模板：** electric_pendulum（复用以圆弧绘制、向心力分析）

**典型题目：**
> 质量为 m 的小球在竖直平面内沿半径为 R 的光滑圆轨道做圆周运动，最高点速度为 v₁，最低点速度为 v₂，求最高点和最低点的压力差。

**场景分镜（12场景）：**

| # | 场景 | 内容 |
|---|------|------|
| 1 | intro | 圆轨道 + 小球 + 已知量标注 |
| 2 | pos_top | 小球在最高点，显示 G、N₁、v₁ |
| 3 | pos_bottom | 小球在最低点，显示 G、N₂、v₂ |
| 4 | sol_q1 | 最高点受力分析 → 求 N₁ |
| 5 | trans | 过渡 |
| 6 | swing | 小球沿轨道运动一周 |
| 7 | highlight | 最高点/最低点高亮标注 |
| 8 | sol_q2 | 最低点受力分析 → 求 N₂ |
| 9 | trans2 | 过渡 |
| 10 | compare | 两侧对比：N₁ vs N₂ |
| 11 | sol_q3 | 求压力差 |
| 12 | ending | 答案汇总 |

**物理公式：**
```
最高点: mg + N₁ = mv₁²/R → N₁ = mv₁²/R − mg
最低点: N₂ − mg = mv₂²/R → N₂ = mv₂²/R + mg
压力差: ΔN = N₂ − N₁ = m(v₂²−v₁²)/R + 2mg
```

**需要新增的 Canvas 绘制：**
- `drawCircularTrack(R, center)` — 圆形轨道（虚线/实线）
- `drawBallOnTrack(x, y, r)` — 在轨道上的小球
- 复用：`drawArrow`（受力箭头）

**参数：**
```json
{
  "m": 0.5, "R": 1.0, "g": 10,
  "v1": 5, "v2": 7
}
```

**预估工时：** 1.5 天
**复用度：** ★★★★☆

### A-2 弹簧振子（简谐振动）

**基于现有模板：** 新写（弹簧绘制是核心新视觉）

**典型题目：**
> 一轻质弹簧一端固定，另一端连接质量为 m 的物块，置于光滑水平面上。弹簧劲度系数为 k，将物块从平衡位置拉开距离 A 后释放，求物块的最大速度、振动周期等。

**场景分镜（10场景）：**

| # | 场景 | 内容 |
|---|------|------|
| 1 | intro | 弹簧+物块+已知量 |
| 2 | force | 受力分析（弹力 F=−kx） |
| 3 | pull | 将物块拉到最大位移 A |
| 4 | release | 释放，物块开始振动 |
| 5 | process | 一个完整周期，速度/位移实时变化 |
| 6 | highlight | 平衡位置速度最大标注 |
| 7 | sol_q1 | 求最大速度 |
| 8 | sol_q2 | 求振动周期 |
| 9 | sol_q3 | 求某位置的速度 |
| 10 | ending | 答案汇总 |

**物理公式：**
```
F = −kx        （胡克定律）
ω = √(k/m)     （角频率）
T = 2π/ω       （周期）
v_max = Aω     （最大速度）
E = ½kA²      （总机械能）
v(x) = √(k(A²−x²)/m)  （任意位置速度）
```

**需要新增的 Canvas 绘制：**
- `drawSpring(x1, y1, x2, y2, coils)` — 弹簧（锯齿/波浪线）
- `drawBlock(x, y, w, h)` — 滑块
- `drawEquilibriumLine(x1, x2, y)` — 平衡位置虚线

**参数：**
```json
{
  "m": 0.5, "k": 100, "A": 0.2, "g": 10,
  "question1": "求最大速度",
  "question2": "求振动周期"
}
```

**预估工时：** 2 天
**复用度：** ★★★☆☆

### A-3 传送带问题

**基于现有模板：** inclined_plane（复用物块绘制、摩擦力方向）

**典型题目：**
> 水平传送带以速度 v₀ = 4 m/s 匀速运动，将一质量为 m = 2 kg 的物块轻放在传送带左端，物块与传送带间动摩擦因数 μ = 0.4，传送带足够长。求物块从放到与传送带共速过程中的位移、摩擦生热等。

**场景分镜（11场景）：**

| # | 场景 | 内容 |
|---|------|------|
| 1 | intro | 传送带+物块+已知量 |
| 2 | force | 受力分析（f = μmg） |
| 3 | start | 物块轻放上传送带 |
| 4 | accelerate | 物块加速（摩擦力做正功） |
| 5 | co_velocity | 共速后匀速 |

| 9 | sol_q3 | 求摩擦生热 |
| 10 | ending | 答案汇总 |

**物理公式：**
```
f = μmg
a = μg
t = v₀ / (μg)
x₁ = v₀²/(2μg)    （加速位移）
Q = μmg·Δx = mg·v₀²/(2μg)·μmg...

```

- `drawConveyor(x, y, w, h)` — 传送带（滚轮+皮带纹理）

**参数：**
```json

```

**复用度：** ★★★★☆

**典型题目：**
> 一质量为 m 的汽车以恒定功率 P 从静止开始在平直路面上运动，受到的阻力恒为 f。求汽车的最大速度、达到某速度所需的时间等。

**场景分镜（10场景）：**

| # | 场景 | 内容 |
|---|------|------|
| 1 | intro | 汽车+路面+已知量 |
| 2 | force | 受力分析（牵引力F vs 阻力f） |
| 3 | start | 汽车启动（P=Fv，v小F大） |
| 4 | accelerate | 加速过程（F逐渐减小） |
| 5 | v_max | 当F=f时速度最大 |

```
P = Fv           （功率）
F − f = ma       （牛顿第二定律）

```

## 3. B组：多物体/多过程

### B-1 动量守恒/碰撞

**基于现有模板：** 新写（双球碰撞是核心新视觉）

**典型题目：**
> 在光滑水平面上，质量为 m₁ = 2 kg 的小球以速度 v₁ = 3 m/s 向右运动，与静止的质量为 m₂ = 1 kg 的小球发生弹性碰撞。求碰撞后两球的速度。

**场景分镜（11场景）：**

| # | 场景 | 内容 |
|---|------|------|
| 1 | intro | 两球+水平面+已知量 |
| 2 | before | 碰撞前（m₁运动，m₂静止） |
| 3 | impact | 碰撞瞬间（弹性/非弹性） |
| 4 | after | 碰撞后（两球各自运动） |
| 5 | compare | 碰撞前后速度对比箭头 |
| 6 | momentum_check | 动量守恒验证 |
| 7 | energy_check | 能量守恒验证（弹性碰撞） |
| 8 | sol_q1 | 求碰撞后速度 |
| 9 | sol_q2 | 求能量损失（非弹性） |
| 10 | ending | 答案汇总 |

**物理公式：**
```
动量守恒: m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
弹性碰撞: ½m₁v₁² + ½m₂v₂² = ½m₁v₁'² + ½m₂v₂'²
弹性碰撞结果:
  v₁' = (m₁−m₂)/(m₁+m₂) · v₁
  v₂' = 2m₁/(m₁+m₂) · v₁
```

**需要新增的 Canvas 绘制：**
- `drawMultiBalls(positions, radius)` — 两个小球
- `drawCollisionEffect(x, y)` — 碰撞特效（闪光）
- `drawMomentumBar(p1, p2)` — 动量柱状图

**参数：**
```json
{
  "m1": 2, "m2": 1, "v1": 3, "v2": 0,
  "elastic": true
}
```

**预估工时：** 2.5 天
**复用度：** ★★☆☆☆

### B-2 板块模型

**基于现有模板：** inclined_plane + 传送带（复用物块绘制和摩擦力）

**典型题目：**
> 质量为 M 的长木板静止在光滑水平面上，质量为 m 的小物块以初速度 v₀ 滑上木板左端，物块与木板间动摩擦因数为 μ，求物块与木板达到共同速度时的位移和相对位移。

**场景分镜（10场景）：**

| # | 场景 | 内容 |
|---|------|------|
| 1 | intro | 木板+物块+已知量 |
| 2 | force | 两者受力分析 |
| 3 | start | 物块滑上木板 |
| 4 | process | 物块减速、木板加速 |
| 5 | co_velocity | 共速后一起运动 |

| 7 | sol_q1 | 求共同速度 |
| 8 | sol_q2 | 求相对位移 |
| 9 | ending | 答案汇总 |

**物理公式：**
```
系统动量守恒: mv₀ = (m+M)v_共
物块: −μmg = ma₁ → a₁ = −μg
木板: μmg = Ma₂ → a₂ = μmg/M
共速时间: t = v₀/(μg(1+m/M))
相对位移: Δx = v₀²/(2μg(1+m/M))
```

- `drawBoard(x, y, w, h)` — 长木板

**参数：**

**场景分镜（10场景）：**

| # | 场景 | 内容 |

### D-2 气体实验定律

**基于现有模板：** 新写（气缸+活塞+P-V图）

**典型题目：**
> 一定质量的理想气体，从状态 A 经历等温过程到状态 B，再经历等容过程到状态 C。已知 pA、VA、TA，求各状态参数。

**场景分镜（9场景）：**

| # | 场景 | 内容 |
|---|------|------|
| 1 | intro | 气缸+活塞+已知量 |
| 2 | state_A | 初始状态标注(p,V,T) |
| 3 | process_AB | 等温过程（活塞移动） |
| 4 | process_BC | 等容过程 |
| 5 | pv_graph | P-V 图绘制 |
| 6 | sol_q1 | 求中间状态参数 |
| 7 | sol_q2 | 求做功/热量 |
| 8 | ending | 答案汇总 |

**物理公式：**
```
pV = nRT (理想气体状态方程)
等温: p₁V₁ = p₂V₂
等容: p₁/T₁ = p₂/T₂
等压: V₁/T₁ = V₂/T₂
W = pΔV (等压做功)
```

**需要新增的 Canvas 绘制：**
- `drawCylinder(x, y, w, h)` — 气缸
- `drawPiston(x, y, w, color)` — 活塞
- `drawPVGraph(points, axes)` — P-V 坐标系

**参数：**
```json
{
  "pA": 2e5, "VA": 3, "TA": 300, "n": 0.5, "R": 8.31
}
```

**预估工时：** 2 天
**复用度：** ★☆☆☆☆

### D-3 机械波

**基于现有模板：** 新写（波形图绘制）

**典型题目：**
> 一列简谐横波沿 x 轴正方向传播，波速 v = 20 m/s。某时刻的波形如图所示，求波的振幅、波长、周期、频率等。

**场景分镜（10场景）：**

| # | 场景 | 内容 |
|---|------|------|
| 1 | intro | 坐标系+波形+已知量 |
| 2 | wave | 波形图标注(λ, A) |
| 3 | propagation | 波向右传播动画 |
| 4 | particle | 某质点振动情况 |
| 5 | highlight | 振动与传播的关系 |
| 6 | sol_q1 | 求波长/振幅 |
| 7 | sol_q2 | 求周期/频率 |
| 8 | sol_q3 | 求某点振动方程 |
| 9 | ending | 答案汇总 |

**物理公式：**
```
v = λf = λ/T
A: 振幅（波形最高点）
λ: 波长（相邻波峰距离）
f = v/λ
T = 1/f = λ/v
```

- `drawWavelength(x1, x2)` — 波长标注
- `drawAmplitude(y1, y2)` — 振幅标注

**参数：**
```json
{
  "v": 20, "lambda": 4, "A": 0.5
}
```

**预估工时：** 2 天
**复用度：** ★☆☆☆☆

### 7.3 常见坑点（从这次教训总结）

| 坑 | 检查方式 |
|----|---------|
| 变量名大小写错误 | 测试时跑一遍，观察控制台错误 |
| 帧循环中抛异常导致动画卡死 | 确保 `requestAnimationFrame` 在 try-catch 外 |
| 画布坐标超出范围 | Canvas 绘制前检查坐标边界 |
| 物理公式错误 | 与标准答案核对 |
| 场景进度计算溢出 | `prog` 值 clamp 到 [0, 1] |
