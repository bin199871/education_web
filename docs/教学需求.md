# 教学需求文档：电场中带电单摆讲解动画

> 基于 [physics-animation-design-philosophy.md](physics-animation-design-philosophy.md) 的设计理念展开的需求分析

---

## 1. 项目概述

### 1.1 项目目标
制作一个全自动播放的物理教学讲解动画，围绕"电场中带电单摆"的经典题目，通过"现象 + 公式"同步呈现的方式，让学生记住物理过程的同时理解公式的物理意义。

### 1.2 题目内容

**已知条件：**
- 匀强电场水平向右，E = 2.0 × 10³ N/C
- 小球质量 m = 0.1 kg，电荷量 q = 5.0 × 10⁻⁴ C（正电）
- 细线长度 L = 1.0 m，不可伸长，绝缘
- 重力加速度 g = 10 m/s²，不计空气阻力
- sin37° = 0.6，cos37° = 0.8

**问题：**
1. 小球所受电场力的大小 F电 和重力与电场力的合力大小 F合
2. 小球运动到细线与竖直方向夹角 θ = 37° 时的速度大小 v
3. 小球运动过程中，细线拉力的最大值 Tmax

**答案：**
1. F电 = qE = 1.0 N, G = mg = 1.0 N, F合 = √(1²+1²) = √2 ≈ 1.41 N
2. h = L(1-cos37°) = 0.2 m, x = L·sin37° = 0.6 m, v = 2√2 ≈ 2.83 m/s
3. 从θ=90°释放到最低点A，vA² = 40, Tmax = 5.0 N

### 1.3 核心设计理念

> **让公式长在现象上** —— 公式在物理现象的时空坐标中自然浮现

设计的四个原则：
1. **现象优先**：动画画面永远是主角，文字公式是配角
2. **因果可视化**：每个物理量变化都要有视觉对应
3. **同步呈现**：公式和现象在同一时空出现，不分离
4. **少即是多**：一次只突出一个核心概念

#### 场景7：Q2解题（56~65%）

- 右侧解题面板滑入
- 主画面小球继续缓慢摆动，保持θ=37°附近

**解题步骤逐行展示：**
```
① 确定初末位置
   小球从最低点A(θ=0)运动到θ=37°
② 计算高度和水平位移
   h = L(1-cos37°) = 1.0×(1-0.8) = 0.2 m
   x = L·sin37° = 1.0×0.6 = 0.6 m
③ 由动能定理求速度
   W合 = F电·x − G·h = 1.0×0.6 − 1.0×0.2
       = 0.6 − 0.2 = 0.4 J
   ½mv² = 0.4 → v² = 0.4×2÷0.1 = 8
   v = √8 = 2√2 ≈ 2.83 m/s
```

**最终答案框脉冲放大：**
```
✅ v = 2√2 ≈ 2.83 m/s
```

#### 场景11：Q3解题（78~87%）

- 右侧解题面板滑入
- 主画面小球在A点附近往复摆动

**解题步骤逐行展示：**
```
① 确定拉力最大位置
   最低点A(θ=0)时速度最大，拉力最大
② 求最低点速度 vA
   从释放点(θ=90°)到最低点A
   W合 = F电·L + G·L = 1.0 + 1.0 = 2.0 J
   ½mvA² = 2.0 → vA² = 40 → vA = √40
③ 求最大拉力
   Tmax − G = mvA²/L
   Tmax = mvA²/L + G = 0.1×40/1.0 + 1.0
        = 4.0 + 1.0 = 5.0 N
```

**最终答案框脉冲放大：**
```
✅ Tmax = 5.0 N
```

## 10. 附注

### 10.1 本题的物理关键点

本题有两个释放过程，是教学中的难点和易错点：

1. **第一次释放（θ=0°→右摆）**：适用于Q1和Q2，从最低点释放，小球在复合场中加速向右摆动，在θ=37°时具有特定速度和能量

2. **第二次释放（θ=90°→最低点）**：适用于Q3，从水平位置释放，摆到最低点时速度最大，拉力最大

通过动画清晰区分这两个过程，能帮助学生理解动能定理在不同物理过程中的应用。

### 10.2 与现有代码的关系

本需求文档基于 `frontend/style-demo.html` 的已有实现进行重构，主要改进：
- 增加几何量可视化标注
- 增加能量柱状图
- 改为画中画解题布局（代替全屏覆盖）
- 增加关键帧自动慢放
- 明确区分两次释放过程
